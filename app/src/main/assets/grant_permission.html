授予剪贴板修改权限

在 Android 10 上必须授予此权限才能开启剪贴板同步，授予方法如下：

  1. 打开手机 “开发者模式” 中的 “USB 调试” 具体操作方法点击此处
  2. 用 USB 连接电脑，此时若弹出调试授权框，请点击 “是”
  3. 在 ADB 目录打开命令提示符，输入 adb shell 具体操作方法点击此处
  4. 执行以下命令（总共三行）： 
    adb -d shell appops set com.kenvix.clipboardsync SYSTEM_ALERT_WINDOW allow

adb -d shell pm grant com.kenvix.clipboardsync android.permission.READ_LOGS

adb shell am force-stop com.kenvix.clipboardsync
